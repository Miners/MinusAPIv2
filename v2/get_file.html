File¶

URL: /api/v2/files/[id]

Attributes:

Name Where? Description
id part of the url File ID
Available methods:
  • GET

    Get Minus File

  • PUT

    Modify Minus File fields

  • DELETE

    Delete Minus File from Folder

Description:
Get, delete or modify a Minus Folder.

PUT fields¶

caption:File caption

Examples¶

Get File¶

Request:

GET /api/v2/files/[file id]?bearer_token=0f513a4a4ce378a72fc4 HTTP/1.1
Host: minus.com
Content-Type: application/x-www-form-urlencoded; charset=UTF-8
Accept: */*
Accept-Charset: UTF-8,*;q=0.5

Delete File¶

Request:

DELETE /api/v2/files/Vafnw8WkWC3b?bearer_token=0f513a4a4ce378a72fc4 HTTP/1.1
Host: minus.com
Content-Type: application/x-www-form-urlencoded; charset=UTF-8
Accept: */*
Accept-Charset: UTF-8,*;q=0.5

Response:

HTTP/1.0 204 NO CONTENT
Vary: Authenticate, Accept, Cookie
Content-Type: application/json
Allow: GET, PUT, DELETE
