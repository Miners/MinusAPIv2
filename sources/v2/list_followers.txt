.. _list_followers:

Follower list
=============


URL: ``/api/v2/users/[slug]/followers``

:slug: User's slug

Example link::
    
    /api/v2/users/api_examples/followers

Description
-----------
    
    This interface lists the follower users of a user. It gives back a list of
    User objects.

.. note::
   This list uses pagination.

Available methods
-----------------

 * GET
    Follower list
