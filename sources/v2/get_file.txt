.. _get_file:

File
====

URL: ``/api/v2/files/[id]``

Attributes:

+------+-----------------+-------------+
| Name | Where?          | Description |
+======+=================+=============+
| id   | part of the url | File ID     |
+------+-----------------+-------------+

Available methods:
 * GET
 * DELETE

Description:
    Gives back the File object of the given ID.
